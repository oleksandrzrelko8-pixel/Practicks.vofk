# Нотатник із тегами

Це консольна програма для збереження та керування нотатками, реалізована на мові Python. Створена в рамках навчальної практики.

## Функціонал програми
- Додавання нотаток із заголовком, текстом та тегами.
- Перегляд списку всіх нотаток.
- Пошук нотаток за певним тегом.
- Детальний перегляд нотатки за її ID.
- Редагування та видалення існуючих нотаток.
- Автоматичне збереження даних у файл `notes.json`.
- Стійкість до некоректного вводу даних користувачем.

## Вимоги до середовища
Для запуску програми потрібен встановлений **Python 3.x**. 
Жодні сторонні бібліотеки не потрібні (використовуються лише вбудовані модулі `json`, `os`, `datetime`).

## Інструкція із запуску (з чистого аркуша)

1. Завантажте всі файли з репозиторію.
2. Відкрийте термінал (або командний рядок) у папці з файлами проєкту.
3. Запустіть програму командою:
   ```bash
   python main.py
   ```
   (Або `python3 main.py` на деяких системах).
4. Дотримуйтесь інструкцій в інтерактивному консольному меню.

## Зберігання даних
При першому збереженні нотатки автоматично створиться файл `notes.json`, в якому будуть зберігатися всі записи у форматі JSON. При наступних запусках програма автоматично підтягне ці дані.
